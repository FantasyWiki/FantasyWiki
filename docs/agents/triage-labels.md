# Triage labels

These are the canonical triage states and the exact label strings to apply in this repository.

| Role | Label |
|---|---|
| needs triage | `needs-triage` |
| needs info | `needs-info` |
| ready for agent | `ready-for-agent` |
| ready for human | `ready-for-human` |
| won't fix | `wontfix` |

## Notes for triage skill

- Apply only these label strings for state transitions.
- If multiple labels are present, prioritize the latest intended triage state.
